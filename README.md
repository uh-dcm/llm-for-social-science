# Large Language Models for Social Sciences

## Learning objectives

* Operational principles of LLMs
* Applications of LLMs for social sciences, including content classification, data generation, social simulation
* Concerns on LLM applications, such as replication and validity

## Before first class

* This course expects basic knowledge of Python or R, for example "Computational thinking, programming and social science."
(Can be taken same time with this course.)
